How to Use the API

  1. Make a GET request to the root endpoint '/' to check if the API is running: 
    GET /

    Response: 'Together Api'

  2. Make a GET request to '/together' with a 'text' query parameter to get a completion based on user input: 
    GET /together?text=YourInputTextHere

    Response: Completion message based on user input
